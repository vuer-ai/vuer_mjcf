# G1 Humanoid Scene

Complete humanoid robot scene with G1.

## Components

- **Robot**: G1 humanoid
- **Environment**: Ground, lighting
- **Control**: Full-body control

## Usage

```python
from vuer_mjcf.stage_sets.g1_scene import G1Scene

scene = G1Scene(assets="path/to/assets")
```

## Use Cases

- Humanoid robot research
- Full-body manipulation
- Bipedal locomotion
- Human-robot interaction
