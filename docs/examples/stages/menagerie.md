# Menagerie

Collection of diverse robots and objects for testing and visualization.

## Components

- Multiple robot models
- Variety of objects
- Ground plane
- Camera rig
- Lighting rig

## Usage

```python
from vuer_mjcf.stage_sets.menagerie import Menagerie

# Load menagerie scene
scene = Menagerie(assets="path/to/assets")
```

## Use Cases

- Robot comparison
- Visualization demos
- Model testing
- Showcase multiple robots
- Educational demonstrations

## Description

The Menagerie scene provides a convenient way to visualize and compare multiple robot models in a single environment. It's useful for demonstrations, testing different robot configurations, and educational purposes.
