# Plate

Tableware plate from Objaverse dataset for manipulation tasks.

## Usage Example

```python
from vuer_mjcf.objects.plate import ObjaverseMujocoPlate

plate = ObjaverseMujocoPlate(name="dinner_plate", pos=[0, 0, 0.8])
```

## Use Cases

- Table setting tasks
- Pick and place
- Stacking demonstrations
