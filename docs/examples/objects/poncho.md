# Poncho

A deformable cloth object using soft body physics.

## Description

The `Poncho` class creates a soft, deformable cloth garment that responds to physics and can drape over objects.

## Usage Example

```python
from vuer_mjcf.objects.poncho import Poncho

poncho = Poncho(name="cloth", pos=[0, 0, 1])
```

## Features

- Deformable soft body
- Cloth dynamics
- Self-collision
- Draping behavior

## Use Cases

- Cloth manipulation research
- Deformable object handling
- Garment manipulation
- Soft body physics testing
