# Ability Hand

Dexterous humanoid hand with 6 DOF for advanced manipulation tasks.

## Usage

```python
from vuer_mjcf.robots.ability_hand import AbilityHandLeft, AbilityHandRight

# Left hand
left_hand = AbilityHandLeft(
    name="left_hand",
    assets="ability_hand"
)

# Right hand
right_hand = AbilityHandRight(
    name="right_hand",
    assets="ability_hand"
)
```
