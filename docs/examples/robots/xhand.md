# XHand

Advanced dexterous hand for complex manipulation tasks.

## Usage

```python
from vuer_mjcf.robots.xhand import XHandLeft, XHandRight

# Left hand
left_hand = XHandLeft(
    name="left_xhand",
    assets="xhand"
)

# Right hand
right_hand = XHandRight(
    name="right_xhand",
    assets="xhand"
)
```
