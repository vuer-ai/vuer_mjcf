# DexHand

Dexterous hand designed for precision manipulation.

## Usage

```python
from vuer_mjcf.robots.dexhand import DexHandLeft, DexHandRight

# Left hand
left_hand = DexHandLeft(
    name="left_dexhand",
    assets="dexhand"
)

# Right hand
right_hand = DexHandRight(
    name="right_dexhand",
    assets="dexhand"
)
```
