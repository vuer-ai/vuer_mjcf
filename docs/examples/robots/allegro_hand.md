# Allegro Hand

Four-fingered dexterous hand for manipulation research.

## Usage

```python
from vuer_mjcf.robots.allegro_hand import AllegroHandLeft, AllegroHandRight

# Left hand
left_hand = AllegroHandLeft(
    name="left_allegro",
    assets="allegro_hand"
)

# Right hand
right_hand = AllegroHandRight(
    name="right_allegro",
    assets="allegro_hand"
)
```
