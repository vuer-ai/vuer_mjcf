# Ufactory xArm7

The xArm7 is a 7-DOF robotic arm from Ufactory with high precision and repeatability.

![Ufactory xArm7 robot](figures/ufactory_xarm7.png)

## Usage

```python
from vuer_mjcf.robots.ufactory_xarm7 import Xarm7
from vuer_mjcf.stage_sets.ufactory_xarm7_scene import UfactoryXArm7Scene

scene = UfactoryXArm7Scene()
```
